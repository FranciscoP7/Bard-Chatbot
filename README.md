# 🤖 Bard - Chatbot de Inglês

**Bard** é uma Inteligência Artificial generativa desenvolvida para auxiliar estudantes e interessados em praticar, aprimorar e evoluir seu inglês de forma prática e interativa. A IA oferece conversas realistas e personalizadas que ajudam no desenvolvimento do idioma, além de outras funcionalidades.

## 📌 Introdução

**Bard** é um projeto de IA focado no aprendizado de inglês. Utilizando recursos avançados da Google API e interfaces modernas, o Bard simula um professor de inglês capaz de responder perguntas, corrigir frases, treinar pronúncia e muito mais.

---

## 🚀 Como Usar

1. **Obtenha uma Google API Key** – instruções no arquivo [`Api Google.md`](./Api%20Google.md).
2. Acesse o Bard pelo site oficial:  
   👉 [iabard.streamlit.app](https://iabard.streamlit.app)

---

## 🧠 Funcionalidades

- Chat em inglês com inteligência artificial.
- Correção gramatical e sugestões de vocabulário.
- Simulação de diálogos do cotidiano.
- Treino de escrita e leitura.
- Aprendizado contextualizado com feedback instantâneo.
